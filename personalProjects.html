Physics Solver




For my senior year independent studies class I built a physics solver in Python. It is around 1030 lines of code and is capable of computing two core physics problems in the AP Physics 1 Exam, the Boat Problem and the Pulley Problem. Since I made the Physics Solver in a non-traditional way using a game engine module instead of a website, I was able to add animations to the objects for a nice visual while challenging and growing my fluency in python and python-related modules.

Link to Python file: https://github.com/RobertP2705/Physics-Solver
